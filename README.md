# 📂 DataVault - Secure File Sharing Platform

**DataVault** is a secure, web-based file-sharing platform built using the **MERN (MongoDB, Express.js, React, Node.js)** stack. It offers enterprise-level security features, ensuring data confidentiality, integrity, and authorized access while mitigating potential threats associated with file sharing in corporate environments.

---

## 🚀 **Project Overview**
**DataVault** addresses modern-day corporate pain points by introducing advanced security features to prevent unauthorized file access and ensure data privacy. It provides a seamless user experience while incorporating robust security protocols, making it ideal for enterprises that prioritize secure data transmission.

---

## 🔐 **Key Features**

### 1. **JWT-Based User Authentication**
- Implements JSON Web Token (JWT) to verify user identity and prevent unauthorized access.
- Ensures secure login and session management.

### 2. **Dynamic Watermarking**
- Embeds unique, invisible watermarks into shared files to track unauthorized redistribution.
- Protects sensitive corporate documents from leaks.

### 3. **Time-Limited URLs**
- Generates secure file links with customizable expiration timers.
- Prevents access to shared files after a defined period.

### 4. **Multi-Layer Authentication (2FA)**
- Requires two or more authentication factors (e.g., password and OTP) to enhance file access security.
- Prevents unauthorized logins even if credentials are compromised.

### 5. **One-Time URL Access**
- Provides secure, single-use URLs that become invalid after one download or viewing session.
- Reduces the risk of multiple unauthorized downloads.

### 6. **Audit Trail & Access Logs**
- Monitors file-sharing activities and maintains logs of user interactions.
- Enables admins to track suspicious activities and maintain audit trails.

## ⚙️ **Tech Stack**

### 📚 **Frontend:**
- **React.js**: Component-based UI framework for building interactive web applications.
- **Redux**: State management for efficient data flow across components.

### 🖥️ **Backend:**
- **Node.js**: JavaScript runtime for server-side logic.
- **Express.js**: Lightweight web framework for RESTful API development.

### 🗂️ **Database:**
- **MongoDB**: NoSQL database for secure and scalable file metadata management.

### 🛡️ **Security:**
- **JWT Authentication**: Validates user sessions and prevents unauthorized access.
